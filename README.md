## Quick Start

```bash
# Clone the repo
$ git clone https://github.com/prefer/virtualenv_example.git

# Run sudo provision
$ sudo ./sudo_provision.sh

# Run provision
$ ./provision.sh

# Run the server
$ workon todo-django
$ cd /vagrant/todomvc-django-backbone/todomvc-django/
$ ./manage.py runserver 0.0.0.0:8000
```
